## Short description

## 紹介文

### 誰が撃ったのか。誰を「本人」として処理するのか。

**Persona Null / 監査室** は、人格・身体・記憶の境界で起きた事件を調査する短編SFアドベンチャーです。

北霞市の都市OSは、義体の所有者、発砲した身体、記録に残る人格署名をひとつの答えにまとめられません。あなたは監査官として端末に接続し、文章で提示された事件概要から、その根拠となる **Memory Network** へ潜ります。

これは、用意された回答を読むだけのゲームではありません。7つの記憶ノードをたどり、記録同士の接続、根拠、矛盾、監査状態を確認し、自分の行政判断を組み立てるゲームです。

### あなたが行うこと

- 記憶ノードを開き、証言・監視記録・人格署名・処理要求を照合する
- 重要な記録を最大3件まで「提出根拠」に登録する
- 記録の食い違いを分類し、必要なら限られた監査リソースで追加解析する
- 条件を満たした後、3つの不可逆な行政処理から最終判断を提出する

### 収録事件

プレイできるのは **Case000「誰が撃ったのか」** です。

人格署名は間宮怜司を示し、発砲した義体は七瀬未織の所有物として登録されている。都市警備局から届いた処理要求に対し、あなたは何を根拠として、誰に責任を結び付けるのかを決めます。

**Case001「焼却されなかった声」は予告表示のみです。プレイ可能な本編は収録していません。**

### 操作

マウスまたはトラックパッドでボタンと Memory Network のノードを選択します。キーボード操作にも対応しています。進行状況と Case000 の結果はブラウザの localStorage に保存されます。

### プレイ環境

- ブラウザでプレイ
- PC ブラウザ推奨
- 狭い画面でも文章は確認できますが、Memory Network の探索はPC向けです
- 音声・音楽なし

**Play:** https://persona-null.vercel.app

## スクリーンショット説明

### 1. タイトル画面

北霞市の監査端末へ接続する入口。仮アクセス権限と Case000 の識別情報が表示され、行政システムを操作する静かな緊張感を提示します。

### 2. Case000 事件概要

発砲事件の争点、間宮怜司と七瀬未織の人物プロフィール、都市警備局からの処理要求を確認する導入画面。最初に「何が起きたか」を読み、次の調査で確かめる問いを整理します。

### 3. Case000 調査画面

7つの記憶ノードを結ぶ Memory Network を探索する中核画面。記録の内容と接続を読み、提出根拠の登録、矛盾分類、追加解析を行いながら最終判断を組み立てます。
